# 1인 창업 손익 시뮬레이터

목표 수입에서 역산하고, 원가구조별 시나리오를 비교하는 인터랙티브 도구

https://chang-hyun-hong.github.io/business-simulator/

## 기능

| 단계 | 기능 | 설명 |
|------|------|------|
| ① 목표 역산 | 연 수입 → 필요 매출/고객수 역산 | 목표 수입, 세율, 이익률, 객단가를 넣으면 "주 몇 명에게 팔아야 하는지" 계산 |
| ② 원가 구조 | 고정비/변동비 분석 | 비용을 입력하면 반도체형(고정비) vs 2차전지형(변동비) 원가 DNA 진단 + BEP 계산 |
| ③ 시나리오 비교 | A/B 원가구조 비교 | 두 가지 비용 구조의 매출 구간별 이익 차이를 테이블 + 차트로 비교 |
| ④ 런웨이 | 생존 가능 기간 계산 | 보유 자금과 매출 성장률로 몇 개월 버틸 수 있는지 시뮬레이션 |
| ⑤ 의사결정 가이드 | 판단 체크리스트 | 고정비형 vs 변동비형 선택 기준, 하이브리드 전략 가이드 |

- 각 결과에 **이미지 저장** 버튼 (PNG 다운로드)
- 서버 없음 — 모든 계산이 브라우저에서 실행, 데이터 저장/전송 없음
