시장 검증 단계

실전 판단 체크리스트

  • BEP까지 버틸 자금이 있는가? — 고정비형은 BEP 도달 전까지 매달 적자. 생존 자금(런웨이) 확인 필수
  • 시장이 검증됐는가? — 아직 수요를 모르면 변동비형으로 먼저 검증, 확인 후 고정비형으로 전환
  • 내 시간이 변동비에 포함되는가? — 시간당 과금 모델은 내 시간 = 변동비. 스케일에 한계가 생김
  • 원가 통제권이 있는가? — 2차전지처럼 원자재가에 연동되면 스프레드를 통제할 수 없음
  • 진입장벽이 있는가? — 없으면 경쟁자 진입 → 마진 축소 (중국의 2차전지 진출 사례)

하이브리드 전략: 단계적 전환

가장 현실적인 1인 창업 전략 1단계: 변동비형으로 시작 → 시장 검증 + 현금흐름 확보
2단계: 검증된 부분을 자동화/시스템화 → 고정비로 전환
3단계: 고정비 레버리지로 이익 극대화

핵심은 "선수금(계약부채)으로 시작"하는 것. 팔리는지 먼저 확인하고 만드는 구조.
